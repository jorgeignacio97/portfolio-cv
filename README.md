# portfolio-cv

Portafolio web personal de Jorge González — Desarrollador Full Stack.

## Stack

- [Next.js 16](https://nextjs.org/) · React 19 · TypeScript
- [Tailwind CSS v4](https://tailwindcss.com/) (CSS-first, `@theme`)
- Paleta: Tokyo Night Dark + Catppuccin Mocha
- Tipografía: Outfit + JetBrains Mono

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).
